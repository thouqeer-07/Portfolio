Welcome to my personal portfolio! This is a modern, responsive web application designed to showcase my journey as an **Artificial Intelligence & Data Science** student and developer. Built with a focus on clean UI, smooth animations, and high performance.

---

## ✨ Features

- **Modern UI/UX**: Designed with a sleek, professional aesthetic using Tailwind CSS.
- **Performant**: Powered by Vite for lightning-fast development and build times.
- **Dynamic Animations**: Enhanced user experience with Framer Motion.
- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Type-Safe**: Developed using TypeScript for robust and maintainable code.
